M5Atom S3R & Arduino Uno R4 Minima - Magnetic Sensor & Servo Control

概要

M5Atom S3R と Arduino Uno R4 Minima を使用し、磁気センサーのデータを送信し、サーボモーターを制御するプロジェクトです。通信方法として「シリアル通信」と「ESP-NOW通信」の2種類があります。

使用ハードウェア

  • M5Atom S3R CAM
  • Arduino Uno R4 Minima
  • サーボモーター （追加予定）

使用ライブラリ

  • M5Unified
  • Servo

参考

  • servoの動きに関して

インストール & 使い方

  1. 必要なライブラリをインストール
  2. M5Atom S3R にコードを書き込み、磁気センサーのデータを送信
  3. Arduino Uno R4 Minima にコードを書き込み、受信データでサーボモーターを制御

シリアル通信

M5Atom S3R から磁気センサーのデータをシリアル通信で送信し、Arduino Uno R4 Minima 側で受信してサーボモーターを制御します。

    Serial.begin(115200);  // ボーレート 115200bps
    Serial.print("磁気センサーの値: ");
    Serial.println(sensorValue);

ESP-NOW通信

ESP-NOW を使い、M5Atom S3R から Arduino Uno R4 Minima にワイヤレスでデータを送信し、サーボモーターを制御します。

    if (esp_now_init() != ESP_OK) {
        Serial.println("ESP-NOW 初期化に失敗しました");
        return;
    }
